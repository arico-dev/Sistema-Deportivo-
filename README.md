Documentación de Requerimientos - Sistema de Gestión Deportiva

Repositorio: Sistema-Deportivo
Tecnologías: HTML, CSS, JavaScript, PHP, MySQL

Descripción

Sistema web para la gestión de actividades deportivas en una institución educativa. Permite administrar usuarios (coordinador, entrenador, estudiante), disciplinas, inscripciones, asistencia, evaluaciones y generar reportes/estadísticas. Diseñado con arquitectura modular (header, sidebar) y pensado para ser responsivo y escalable.

Características principales

Gestión de usuarios: registro, edición, eliminación y roles (Coordinador, Entrenador, Estudiante).

Gestión de disciplinas: creación, edición, eliminación y asignación de entrenadores.

Inscripciones: inscribir estudiantes en disciplinas, activar/desactivar y consultas por filtros.

Control de asistencia: marcar presencia/ausencia/retardo, historial y reportes.

Evaluaciones: registro de puntuaciones, comentarios y consultas históricas.

Dashboard con métricas y generación de reportes (asistencia y rendimiento).

Requerimientos (mínimos)

PHP 7.4+ (o PHP 8+)

Servidor web (Apache / Nginx)

MySQL 5.7+ / MariaDB

Extensiones PHP: pdo, pdo_mysql, mysqli, mbstring, json

Navegador moderno (Chrome, Firefox, Edge)
